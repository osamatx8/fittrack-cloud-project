## Overview

FitTrack is a cloud-based fitness tracking website designed to help users log workouts, calculate training volume, and access fitness resources from anywhere.  
The platform includes:

- **Workout Log System (PHP backend + file storage)**
- **Workout Volume Calculator (JavaScript)**
- **YouTube Fitness Search Tool (PHP)**
- **Full domain integration with SSL**
- **Live cloud hosting via Azure**

This project demonstrates real cloud deployment, DNS setup, HTTPS configuration, and backend scripting on an IaaS server.


## Technology Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Server:** Apache on Ubuntu 24.04  
- **Hosting:** Microsoft Azure (IaaS Virtual Machine)  
- **Domain:** Namecheap → DNS hosted on Azure (`fittrack.world`)  
- **Security:** Let’s Encrypt SSL (HTTPS enabled)
