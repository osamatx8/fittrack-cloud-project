About This Project

FitTrack is a cloud-based fitness platform hosted on Microsoft Azure. It provides simple, accessible tools for logging workouts, calculating training volume, and exploring fitness video content.

The website runs on an Ubuntu Server with Apache, using HTML, CSS, JavaScript, and PHP. It demonstrates cloud hosting, server setup, DNS configuration, and backend scripting.

The project includes:

  • Workout logging system with saved entries
  • A workout summary generator
  • YouTube fitness video search tool
  • JavaScript-based volume calculator

This page provides a simple overview of the purpose and structure of the FitTrack platform.
